The native interface includes a built-in calculator that evaluates mathematical expressions as you type.

<Aside type="note">
The calculator is only available in **native mode**. It is not available when using Fuzzel mode.
</Aside>

## Overview

Simply type a mathematical expression in the launcher, and the calculator will automatically detect and evaluate it. Results can be copied to the clipboard using Enter.

</TabItem>
<TabItem label="Subtraction">

### Mathematical Functions

The calculator supports common mathematical functions:

| Function | Description | Example |
|----------|-------------|--------|
| `sqrt(x)` | Square root | `sqrt(16)` → 4 |
| `sin(x)` | Sine (radians) | `sin(0)` → 0 |
| `cos(x)` | Cosine (radians) | `cos(0)` → 1 |
| `tan(x)` | Tangent (radians) | `tan(0)` → 0 |
| `log(x)` | Base-10 logarithm | `log(100)` → 2 |
| `ln(x)` | Natural logarithm | `ln(2.718)` → ~1 |
| `exp(x)` | Exponential (e^x) | `exp(1)` → ~2.718 |
| `abs(x)` | Absolute value | `abs(-5)` → 5 |
| `floor(x)` | Round down | `floor(3.7)` → 3 |
| `ceil(x)` | Round up | `ceil(3.2)` → 4 |

### Complex Expressions

You can combine multiple operations and functions:

<Tabs>
<TabItem label="Nested Functions">
```text
sqrt(16) + abs(-5)
# Result: 9
```
</TabItem>
<TabItem label="Order of Operations">
```text
2 + 3 * 4
# Result: 14 (multiplication first)
```
</TabItem>
<TabItem label="Parentheses">
```text
(2 + 3) * 4
# Result: 20
```
</TabItem>
<TabItem label="Scientific">
```text
exp(ln(10))
# Result: 10
```
</TabItem>
</Tabs>

## Usage

### Basic Usage

1. Launch Raffi in native mode
2. Type a mathematical expression
3. The result appears automatically
4. Press **Enter** to copy the result to clipboard

![Calculator example](../../../assets/screenshots/calculator.png)

### Keyboard Shortcuts

- **Enter**: Copy result to clipboard
- **Esc**: Clear input and return to normal mode

## Configuration

The calculator is enabled by default in native mode. To disable it:

```yaml raffi.yaml
addons:
  calculator:
    enabled: false
```

### Enable Calculator

<Tabs>
  <TabItem label="Enabled (Default)">
    ```yaml
    addons:
      calculator:
        enabled: true
    ```
  </TabItem>
  <TabItem label="Disabled">
    ```yaml
    addons:
      calculator:
        enabled: false
    ```
  </TabItem>
</Tabs>

## Requirements

<Aside type="note">
The calculator requires `wl-copy` (from wl-clipboard) to copy results to the clipboard.
</Aside>

Install wl-clipboard on your system:

<Tabs>
<TabItem label="Arch Linux">
```bash
sudo pacman -S wl-clipboard
```
</TabItem>
<TabItem label="Debian/Ubuntu">
```bash
sudo apt install wl-clipboard
```
</TabItem>
<TabItem label="Fedora">
```bash
sudo dnf install wl-clipboard
```
</TabItem>
</Tabs>

## Tips

<Aside type="tip">
You don't need to type a special prefix or keyword. Just start typing a mathematical expression and the calculator will activate automatically.
</Aside>

<Aside type="caution">
Make sure you have `wl-copy` installed if you want to copy results to the clipboard. Without it, you can still see the results but cannot copy them.
</Aside>
